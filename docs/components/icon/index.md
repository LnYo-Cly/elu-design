## 基础使用

可以通过`size`、`name`、`color`属性去修改图标。

## API

| 属性名 | 说明     | 类型          | 默认值 |
| ------ | -------- | ------------- | ------ |
| size   | 大小     | number/string | -      |
| name   | 图标名称 | string        | -      |
| color  | 图标颜色 | string        | -      |
